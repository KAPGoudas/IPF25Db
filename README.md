# IPF25Db
Στο συγκεκριμένο αποθετήριο βρίσκονται όλα τα αρχεία που χρησιμοποιήθηκαν στη βάση δεδομένων IPF25Db και την εφαρμογή της για τη Μεταπτυχιακή Διπλωματική Εργασία με τίτλο "Προστασία Πνευματικών Δικαιωμάτων μέσω Ψηφιακής Βάσης Δεδομένων" του φοιτητή Κωνσταντίνου-Αναστάσιου Γούδα για το ΔΠΜΣ "Υπολογιστική Δεδομένων και Αποφάσεων" του Πανεπιστημίου Πατρών.

Η βάση αυτή θα μπορούσε να χρησιμοποιηθεί και ως υπόβαθρο για περεταίρω έρευνα στον τομέα της προστασίας των πνευματικών δικαιωμάτων.
Για αυτόν το λόγο κυκλοφορεί με την άδεια MIT που επιτρέπει την αντιγραφή του κώδικα, καθώς και την τροποποίηση του αντιγράφου.
Ωστόσο, για την ορθή λειτουργία του κώδικα, είναι απαραίτητο να ληφθούν υπόψιν ορισμένες παράμετροι:

1) Το αρχείο conn.php μπορεί να λειτουργεί κανονικά σε ένα server τύπου localhost, αλλά ενδεχομένως σε ένα server να μην ισχύει αυτό. Ίσως χρειαστεί, με άλλα λόγια, η σύνδεση των απαραίτητων PHP αρχείων με τη βάση να γίνει στο καθένα ξεχωριστά.
