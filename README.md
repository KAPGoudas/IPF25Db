# IPF25Db
Στο συγκεκριμένο αποθετήριο βρίσκονται όλα τα αρχεία που χρησιμοποιήθηκαν στη βάση δεδομένων IPF25Db και την εφαρμογή της για τη Μεταπτυχιακή Διπλωματική Εργασία με τίτλο "Προστασία Πνευματικών Δικαιωμάτων μέσω Ψηφιακής Βάσης Δεδομένων" του φοιτητή Κωνσταντίνου-Αναστάσιου Γούδα για το ΔΠΜΣ "Υπολογιστική Δεδομένων και Αποφάσεων" του Πανεπιστημίου Πατρών.

Η βάση αυτή θα μπορούσε να χρησιμοποιηθεί και ως υπόβαθρο για περεταίρω έρευνα στον τομέα της προστασίας των πνευματικών δικαιωμάτων.
Για αυτόν το λόγο κυκλοφορεί με την άδεια MIT που επιτρέπει την αντιγραφή του κώδικα, καθώς και την τροποποίηση του αντιγράφου.
Ωστόσο, για την ορθή λειτουργία του κώδικα, είναι απαραίτητο να ληφθούν υπόψιν ορισμένες παράμετροι:

1) Το αρχείο της βάσης δεδομένων κατασκευάστηκε στο πρόγραμμα MySQL. Προτείνεται, λοιπόν, η βάση που θα χρησιμοποιηθεί στηριζόμενη στο αρχείο αυτό να είναι επίσης σε MySQL. Φυσικά, η απόφαση για το πρόγραμμα που θα χρησιμοποιηθεί για την κατασκευή της βάσης ανήκει αποκλειστικά στον ίδιο το χρήστη.
2) Η εφαρμογή PHP δοκιμάστηκε στο πρόγραμμα XAMPP. Εφόσον ο χρήστης επιθυμεί να χρησιμοποιήσει το πρόγραμμα XAMPP, και με την προϋπόθεση ότι έχει μια, έστω και στοιχειώδη, εξοικείωση με το XAMPP, οφείλει να τοποθετήσει το φάκελο στο htdocs του XAMPP.
3) Το αρχείο conn.php μπορεί να λειτουργεί κανονικά σε ένα server τύπου localhost, αλλά ενδεχομένως σε ένα server να μην ισχύει αυτό. Ίσως χρειαστεί, με άλλα λόγια, η σύνδεση των απαραίτητων PHP αρχείων με τη βάση να γίνει στο καθένα ξεχωριστά.
4) Η αλλαγή των στοιχείων που αναφέρονται στο conn.php είναι υποχρεωτική. Οι παράμετροι $database, $servername, $username και $password οφείλουν να φέρουν τα ονόματα της καινούργιας βάσης, του server όπου θα δημοσιευθεί η βάση, του ονόματος χρήστη και του κωδικού της βάσης αντίστοιχα.
5) Για λόγους ασφαλείας, είναι απαραίτητη η αλλαγή στον κωδικό του pass.php, ή και η πλήρης αλλαγή του κώδικα του αρχείου.

Αυτές είναι όλες οι σημαντικές παράμετροι. Καλή συνέχεια!
