# IPF25Db
Παρακάτω βρίσκονται όλα τα αρχεία που χρησιμοποιήθηκαν στη βάση δεδομένων IPF25Db και την εφαρμογή της για τη Μεταπτυχιακή Διπλωματική Εργασία με τίτλο "Προστασία Πνευματικών Δικαιωμάτων μέσω Ψηφιακής Βάσης Δεδομένων" του φοιτητή Κωνσταντίνου-Αναστάσιου Γούδα για το ΔΠΜΣ "Υπολογιστική Δεδομένων και Αποφάσεων" του Πανεπιστημίου Πατρών
